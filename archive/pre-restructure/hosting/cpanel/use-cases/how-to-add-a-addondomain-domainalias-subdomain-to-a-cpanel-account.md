---
title: "How to add a addondomain/domainalias/subdomain to a cPanel account?"
---



<div class="api-io api-io--command"><span class="api-io__label">Command</span>
<pre class="api-io__code"><code>COMMAND = AddHosting
CLASS = cpanel
TYPE = addondomain|domainalias|subdomain
ACCOUNT = cPanelAccountIDofEndcustomer
OBJECT = subdomain.my-first-cpanel-account.com</code></pre></div>
