**Note**

Please use hostnames when connecting to our API gateways instead of IP addresses.

The IP addresses of our hosts can change from time to time, resulting in connection issues when used to connect. Our hostnames will not change and using them ensures proper connection to our API. You can find all hostnames and TCP/IP ports of our API gateways within our [control panel](https://account.centralnicreseller.com/api/gateways/) in the top menu at "API" -> "API Gateways".
