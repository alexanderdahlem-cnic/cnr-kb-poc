Our mregd Server for Unix / Linux systems opens a persistent SSL connection through which you can connect directly on the command line. This server handles all connections with high performance and security. After you installed the server you can also access via PHP and develop your own scripts. The software can be downloaded in the CentralNic Reseller control panel. You will find example - scripts inside the package, too.

**Note**

Please see CentralNic Reseller control panel for connection details of the MREG gateway.
