The following API commands are available for premium domains.

[AddDomain](../../api/api-command/AddDomain#premium)

Initiates the domain name registration at the registry

[CheckDomain](../../api/api-command/CheckDomain#premium)

Check availability of a single domain

[CheckDomains](../../api/api-command/CheckDomains#premium)

Check availability of multiple domains

[QueryRegisteredPremiumDomainList](../../api/api-command/QueryRegisteredPremiumDomainList#premium)

Query list of registered premium domain names in account

[RenewDomain](../../api/api-command/RenewDomain#premium)

Command used to renew domains explicitly for a specified period of time

[StatusDomain](../../api/api-command/StatusDomain#premium)

Gather information about a domain name in account

[TransferDomain](../../api/api-command/TransferDomain#premium)

This command allows you to request, approve, deny or cancel a domain transfer.

**Note for EU members**

EU members without valid VAT ID and German customers, will receive the response of the domain fee including VAT.
