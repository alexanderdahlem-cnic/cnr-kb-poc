The following API commands are available for premium domains.

| [AddDomain](../../../api/api-command/adddomain/#premium) | Initiates the domain name registration at the registry |
| --- | --- |
| [CheckDomain](../../../api/api-command/checkdomain/#premium) | Check availability of a single domain |
| [CheckDomains](../../../api/api-command/checkdomains/#premium) | Check availability of multiple domains |
| [QueryRegisteredPremiumDomainList](../../../api/api-command/queryregisteredpremiumdomainlist/#premium) | Query list of registered premium domain names in account |
| [RenewDomain](../../../api/api-command/renewdomain/#premium) | Command used to renew domains explicitly for a specified period of time |
| [StatusDomain](../../../api/api-command/statusdomain/#premium) | Gather information about a domain name in account |
| [TransferDomain](../../../api/api-command/transferdomain/#premium) | This command allows you to request, approve, deny or cancel a domain transfer. |

**Note for EU members**

EU members without valid VAT ID and German customers, will receive the response of the domain fee including VAT.
