The following API commands are available for premium domains.

:::command[Command]

```text
AddDomainInitiates the domain name registration at the registryCheckDomainCheck availability of a single domainCheckDomainsCheck availability of multiple domainsQueryRegisteredPremiumDomainListQuery list of registered premium domain names in accountRenewDomainCommand used to renew domains explicitly for a specified period of timeStatusDomainGather information about a domain name in accountTransferDomainThis command allows you to request, approve, deny or cancel a domain transfer.
```

:::

**Note for EU members**

EU members without valid VAT ID and German customers, will receive the response of the domain fee including VAT.
