The following EPP commands are available for premium domains.

:::command[Command]

```text
Domain:CheckCheck availability of a single domainDomain:CreateInitiates the domain name registration at the registryDomain:InfoGather information about a domain name in accountDomain:RenewCommand used to renew domains explicitly for a specified period of timeDomain:RestoreCommand used to restore a domainTransfer:RequestInitiates the domain transfer
```

:::
