The following EPP commands are available for premium domains.

[Domain:Check](../../api/epp-command/Domain_Check#premium)

Check availability of a single domain

[Domain:Create](../../api/epp-command/Domain_Create#premium)

Initiates the domain name registration at the registry

[Domain:Info](../../api/epp-command/Domain_Info#premium)

Gather information about a domain name in account

[Domain:Renew](../../api/epp-command/Domain_Renew#premium)

Command used to renew domains explicitly for a specified period of time

[Domain:Restore](../../api/epp-command/Domain_Restore#premium)

Command used to restore a domain

[Transfer:Request](../../api/epp-command/Transfer_Request#premium)

Initiates the domain transfer
