:::command[ACL API-Commands]

```text
ACL API-CommandsThe Access Control List specifies which actions on certain objects a user is allowed to conduct.AddAclAdd location rights for a given user or group.AddUserCreates a user or a group.DeleteAclRemove location rights for a given user or group.DeleteUserDeletes a user or a group.ModifyAclModify location rights for a given user or group.ModifyUserModify already existing ACL userQueryACLListShows all locations with granted rights of a given user or group.QueryUserListShows a list of all users or groups.StatusAclShows granted rights for a specific location of a given user or group.StatusUserShows the status of a given user or group.
```
