Remove location "domain":

:::command[Command]

```text
command = DeleteAcl
username = user1
location = DOMAIN
```

:::

### Attributes

| **username** | (TEXT) - Mandatory. "username" is either name of a user or a group.  
 |
| --- | --- |
| **location** | (TEXT) - Mandatory; see Locations list. |
