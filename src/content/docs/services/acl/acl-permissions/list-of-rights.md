---
title: "List of Rights"
---

The following rights can be applied to each location:

**ADD**

Add an object.

**CHECK**

Check an object.

**DELETE**

Delete an object.

**MODIFY**

Modify an object and its parameters.

**QUERY**

Query, usually for lists.

**RENEW**

Renew an object.

**STATUS**

Retrieve the status of an object.

**TRANSFER**

Transfer an object, in particular domains.
