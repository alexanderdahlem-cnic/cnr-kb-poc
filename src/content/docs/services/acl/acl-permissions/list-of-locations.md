The following Locations are available:

| **ACCOUNT** | All account related actions. Special right: CHARGE |
| --- | --- |
| **ACL** | Manage ACL rights |
| **CERTIFICATE** | SSL Certificates |
| **CONTACT** | Contact Handles |
| **DNS** | KeyDNS DNS zone files |
| **DOMAIN** | Domains |
| **EVENT** | Event handling |
| **FTP** | FTP access |
| **LOG** | Log access |
| **NAMESERVER** | Name server and host objects |
| **PRICE** | All pricing related actions |
| **REGISTRAR** | Sub-Reseller management |
| **SERVER** | vServer and Root Server |
| **SMS** | SMS Services. Special rights: CHARGE, SEND |
| **SUPPORT** | Support |
| **TOOL** | Tools |
| **TRADEMARK** | Trademark Clearing House records |
| **WEBHOSTING** | Webspaces, Databases, Mailspaces |
